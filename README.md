# 🇮🇳 Bazaar Mood — Indian Market Dashboard

## Overview
Bazaar Mood is a comprehensive Indian stock market intelligence dashboard providing real-time data, options analytics, sentiment analysis, sector tracking, and technical support/resistance levels.

## Project Structure
```
Bazaar Mood/
├── frontend/          # UI — charts, dashboards, components
├── backend/           # Core engines and API layer
│   ├── data_engine/       # Market data ingestion & processing
│   ├── option_engine/     # Options chain analysis & greeks
│   ├── sentiment_engine/  # News & social sentiment analysis
│   ├── sector_engine/     # Sector rotation & heatmaps
│   └── support_engine/    # Support & resistance level detection
├── database/          # DB schemas, migrations, seeds
├── docs/              # Architecture docs & API references
├── logs/              # Application & error logs
└── tests/             # Unit, integration & E2E tests
```
