# 🧪 Tests

## Overview
All automated tests for Bazaar Mood — unit, integration, and end-to-end.

## Test Structure (Planned)
```
tests/
├── unit/           # Isolated unit tests per engine
├── integration/    # Cross-engine & API integration tests
└── e2e/            # End-to-end browser tests (Playwright)
```

## Planned Tools
- **pytest** — Python backend unit & integration tests
- **Playwright** — E2E frontend testing
- **pytest-cov** — Code coverage reporting
