# 📈 Option Engine

## Overview
Analyses NSE F&O options chain data to surface actionable insights including Open Interest (OI) buildup, PCR, max pain, and Greeks (Delta, Gamma, Theta, Vega).

## Responsibilities
- Options chain parsing (NSE API)
- Open Interest analysis & OI change tracking
- Max Pain calculation
- Put-Call Ratio (PCR) computation
- Greeks calculation (Black-Scholes)
- IV (Implied Volatility) surface generation

## Planned Modules
- `chain_parser.py` — Fetches & parses live NSE options chain
- `greeks.py` — Black-Scholes Greeks calculator
- `max_pain.py` — Max pain price computation
- `oi_analyzer.py` — OI trends, buildup/unwinding detection
- `iv_surface.py` — Implied volatility surface builder
