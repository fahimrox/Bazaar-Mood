# 🏭 Sector Engine

## Overview
Tracks Indian market sector performance, rotation signals, and generates real-time heatmaps across NSE indices and SEBI-defined sectors.

## Responsibilities
- Sector-wise performance computation (NIFTY sectoral indices)
- Sector rotation signal generation (relative strength)
- Intraday & multi-timeframe sector heatmap data
- FII/DII sector-wise flow tracking
- Top gainers/losers per sector

## Sectors Tracked
IT • Banking & Finance • Pharma • Auto • FMCG • Energy • Metals • Realty • Media • Infrastructure • PSU • Defence

## Planned Modules
- `sector_mapper.py` — Maps stocks to SEBI/NSE sectors
- `heatmap_builder.py` — Generates heatmap-ready data payloads
- `rotation_signal.py` — Relative strength rotation analysis
- `flow_tracker.py` — FII/DII institutional flow aggregation
