# 📏 Support Engine

## Overview
Detects and tracks key support and resistance levels for Indian stocks and indices using price action, volume profile, and technical indicator analysis.

## Responsibilities
- Pivot point calculation (Classic, Camarilla, Woodie, Fibonacci)
- Volume Profile & Point of Control (POC) detection
- Historical swing high/low level mapping
- Dynamic support/resistance using moving averages
- Breakout & breakdown alert generation

## Planned Modules
- `pivot_calculator.py` — Multi-type pivot point computation
- `volume_profile.py` — Volume-at-price & POC analysis
- `swing_detector.py` — Fractal-based swing high/low detection
- `alert_engine.py` — Price proximity alert triggers
- `fibonacci.py` — Fibonacci retracement/extension levels
