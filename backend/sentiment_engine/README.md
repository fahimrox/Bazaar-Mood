# 🧠 Sentiment Engine

## Overview
Aggregates and scores market sentiment from financial news, social media, and analyst reports using NLP models tailored for Indian financial markets.

## Responsibilities
- Financial news scraping (MoneyControl, ET Markets, NSE announcements)
- Social media monitoring (Twitter/X, Reddit r/IndiaInvestments)
- NLP-based sentiment scoring (Positive / Neutral / Negative)
- Entity extraction (Stock name, sector, event type)
- Sentiment trend aggregation per symbol / sector

## Planned Modules
- `scraper.py` — News & social media data collectors
- `nlp_scorer.py` — Sentiment scoring with FinBERT / custom model
- `entity_extractor.py` — Named entity recognition for tickers
- `aggregator.py` — Rolls up sentiment scores by symbol & sector
