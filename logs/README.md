# 📋 Logs

## Overview
Runtime logs generated by backend engines, API servers, and scheduled jobs.

## Log Types
| File Pattern              | Description                        |
|---------------------------|------------------------------------|
| `app_YYYY-MM-DD.log`      | General application logs           |
| `error_YYYY-MM-DD.log`    | Error and exception traces         |
| `data_engine_*.log`       | Market data ingestion logs         |
| `scheduler_*.log`         | Cron job / task scheduler logs     |

## Notes
- Log files are **gitignored** by default.
- Use a log rotation policy (e.g., daily, max 30 days).
- In production, logs should be shipped to a centralized service (e.g., Loki, CloudWatch).
