# 🗄️ Database

## Overview
Contains all database schemas, migration scripts, seed data, and query utilities for Bazaar Mood.

## Planned Databases
- **TimescaleDB / PostgreSQL** — OHLCV time-series market data
- **Redis** — Real-time caching & pub/sub for live prices
- **SQLite** — Lightweight local development store

## Structure (Planned)
```
database/
├── migrations/    # Versioned schema migrations
├── seeds/         # Sample/seed data for dev/testing
└── queries/       # Reusable SQL query templates
```
